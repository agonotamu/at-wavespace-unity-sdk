# 🎧 AT WaveSpace — Unity WFS Audio SDK

High-precision 3D audio engine for Unity powered by Wave Field Synthesis.

---

## ✨ Overview

AT WaveSpace is a native spatial audio engine based on **Wave Field Synthesis (WFS)**.

It enables physically grounded sound field reproduction inside Unity using a high-performance C++ DSP core.

Built with:

- JUCE
- Native DSP
- Research from CNRS LMA
- Real-time spatial rendering
